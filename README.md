# AGENTS.md

My cross-project operating contract for AI coding agents (Claude Code,
GitHub Copilot, Cursor, and others). See [`AGENTS.md`](AGENTS.md) for the
contract itself.

## How I use it

I drop `AGENTS.md` into the root of every project I work on. AI tools that
read agent contracts pick it up automatically.

## Updates

This file is identical across all my projects. When a genuinely
cross-project rule emerges, I edit it here first and then propagate the
edit to each project root.
